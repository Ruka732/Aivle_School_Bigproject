VITA 소개
---
사용자가 보유 중인 영상을 기반으로 AI 채팅을 제공하는 서비스</BR>
사용자가 영상을 업로드하면 AI가 내용을 요약하여 사용자에게 보여준 후 영상 내용을 기반으로 대화를 주고 받을 수 있는 챗봇
